```txt
Every Game Boy ROM starts with exactly the same header structure:
┌─────────────────────────────────────┐
│ 0x000-0x03F: Nintendo Logo (256B)   │
│ 0x040-0x133: Title/Manufacturer     │
│ 0x134-0x143: Game Title (16 chars)  │ ← HEADER STARTS
│ 0x144-0x145: Manufacturer Code      │
│ 0x146: CGB Flag                     │
│ 0x147: Cartridge Type               │ ← TELLS YOU MBC TYPE
│ 0x148: ROM Size                     │ ← TELLS YOU ROM SIZE
│ 0x149: RAM Size                     │ ← TELLS YOU RAM SIZE
│ 0x14A: Destination Code             │
│ 0x14B: Old Licensee Code            │
│ 0x14C: Mask ROM Version             │
│ 0x14D: Header Checksum              │
│ 0x14E-0x14F: Global Checksum        │
│ 0x150+: ACTUAL GAME CODE BEGINS     │ ← GAME CODE STARTS
└─────────────────────────────────────┘
```
