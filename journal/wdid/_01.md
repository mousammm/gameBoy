#### Day2 Cartridge type

* created the mmu so after checking the cart type 
* loaded the cart->data to memory[0x0000-0x7FFF]
* well for now i only did the 0x00 type 
* also i did for read and write so far as its only 0x00 it was eazy

```c
// Factory function in mmu_init()
switch(cart->cartridge_type) {
    case 0x00: mmu->mbc = mbc_none_create(cart); break;
    case 0x01: mmu->mbc = mbc1_create(cart); break;
    case 0x02: mmu->mbc = mbc1_ram_create(cart); break;
    case 0x03: mmu->mbc = mbc1_battery_create(cart); break;
    case 0x05: mmu->mbc = mbc2_create(cart); break;
    case 0x06: mmu->mbc = mbc2_battery_create(cart); break;
    case 0x0F: case 0x10: case 0x11: case 0x12: case 0x13:
        mmu->mbc = mbc3_create(cart); break;
    case 0x19: case 0x1A: case 0x1B: case 0x1C: case 0x1D: case 0x1E:
        mmu->mbc = mbc5_create(cart); break;
    default: mmu->mbc = mbc_none_create(cart); break; // Fallback
}
```
